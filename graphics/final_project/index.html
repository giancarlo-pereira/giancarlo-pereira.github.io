Welcome to ESCAPE THE MAZE, an a-maze-ing game!

The goal of this game is to escape a maze.
Choose a DIFFICULTY:
- Peaceful has unlimited powers and no timer.
- Medium has a timer and all powers.
- Hard disables all powers.
- Debug has unlimited powers, but no textures.

Choose the SIZE of the maze.
The bigger it is, the harder it will be to leave.

Once you click START, the game will generate a random
maze and you will start in random place.

GOOD LUCK!
