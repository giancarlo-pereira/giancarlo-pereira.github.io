CLICK to add a new point.
CLICK on an existing point to select it (the color changes to green).
ESCAPE to unselect a point (the color changes back to gray).
CLICK AND DRAG a point to move it around.
BACKSPACE a selected point to remove it.
R to remove all points.
ENTER to generate the spline.
